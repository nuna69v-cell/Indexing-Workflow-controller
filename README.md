# MQL5-Google-Onedrive

MQL5 Google OneDrive integration for MetaTrader 5.

## Overview

This project provides integration between MQL5 (MetaTrader 5) and Google OneDrive for file synchronization and backup.

## Original Repository

- **Source**: https://github.com/A6-9V/MQL5-Google-Onedrive.git
- **Status**: Merged into Workspace Monorepo

## Project Structure

```
MQL5-Google-Onedrive/
├── src/           # MQL5 source files (.mq5, .mqh)
├── include/       # Header files
├── docs/          # Documentation
├── tests/         # Test files
└── README.md       # This file
```

## Requirements

- MetaTrader 5
- MQL5 compiler
- Google OneDrive API credentials

## Setup

1. Configure Google OneDrive API credentials
2. Set up MQL5 environment
3. Compile and install EA/Indicator
